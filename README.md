# Provision Kali Metasploitable VMs using Hashicorp Vagrant
- Install Vagrant
- Oracle Virtualbox

To start
```angularjs
vagrant up
```

Check status 
```angularjs
vagrant status
```

To destroy
```angularjs
vagrant destroy
```
